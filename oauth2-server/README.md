请求授权
====
### 授权地址，获取code
```
http://localhost:9001/oauth/authorize?response_type=code&scope=pwd&client_id=acme&state=09876999
```
### 登录
#### 回调并返回code
```
http://localhost:9001/?code=eVeJ6G&state=09876999
```
